Structure Beneficiary.​URI 

@available(*, deprecated, message: "Use BeneficiaryAccount instead.") public struct URI: Equatable, ExpressibleByStringLiteral

A type representing the URI for receiving transfers.

A URI is composed with two parts, a kind with value of e.g. iban and an account number.
